# homebrew-tools

Homebrew formulae for Claude Code tools.

## Install

```bash
brew tap rydersd/tools
brew install claude-sync
```

## Formulae

| Formula | Description |
|---------|-------------|
| `claude-sync` | Sync Claude Code configuration (skills, agents, rules, hooks) between machines via git |

## Updating

```bash
brew update
brew upgrade claude-sync
```
